# 決算カード照合ガイダンス

## 基本方針

本リポジトリは、任意の決算カードXLSXから正確に数字を読む汎用抽出器を提供しない。版面、年度、自治体によってセル配置が動き、ラベル近傍の数値が対象値なのか年度見出しなのかを一般的な規則では区別できない。比較は利用者のローカル環境で、AI、人手、表計算ソフトなどを組み合わせて行う。

共通化するのは、安定した索引から主経路と同じ会計年度の市町村決算カードページを発見し、対象都道府県のXLSXを取得して、Content-TypeとXLSX署名を検証し、SHA-256、取得時刻、キャッシュ位置、解決済みURLを記録することである。`--cross-check` は照合資料を準備するが、決算カードから値を読まず、概況XLSXとの一致・不一致を判定しない。

## 比較AIへの依頼方針

比較AIには、分析や判断ではなく、指定した自治体と指標のセル特定、転記、差異の列挙だけを依頼する。

```text
次の概況XLSXの抽出結果と、ローカルに保存した同年度・同自治体の決算カードXLSXを比較してください。

禁止:
- ラベルに最も近い数値を根拠なく採用しない
- 年度見出しを指標値として扱わない
- 読めない数字や欠測記号を推測しない
- 単位を勝手に換算しない
- 差異を解消するために主経路の値を上書きしない

必須:
- 自治体名、会計年度、指標名、単位を両資料で確認する
- 決算カード側のシート名とセル位置を記録する
- 両資料の原表記を残す
- 一致、不一致、比較不能を指標ごとに報告する
- 読み取れないセルや定義差は要確認として報告する
```

## 人が先に決めること

比較前に、人が次を指定する。

1. 対象自治体と会計年度。
2. 比較する指標。
3. 主経路とする概況XLSXの抽出結果。
4. 両資料の単位と会計範囲。
5. 決算カード内で対象自治体を確認する方法。
6. 比較結果を記録するノートまたは台帳。

AIへ都道府県単位のXLSXを丸ごと渡して「6指標を自動検算して」と依頼しない。自治体と指標を小さく区切り、シート名とセル位置を人が追える形にする。

## 比較後の検査

1. 決算カードが主経路と同じ会計年度・自治体の資料か確認する。
2. 指標名、定義、会計範囲、単位が比較可能か確認する。
3. 決算カード側のシート名、セル位置、原表記を記録する。
4. 一致、不一致、比較不能を指標ごとに記録する。
5. 不一致があれば、片方に合わせて数字を直さず、両方の原典へ戻る。
6. 確認後、証拠台帳または判断ノートへ、比較日、対象資料、差異、未解決事項を記録する。

## 一致しないとき

決算カードとの差異は、概況XLSXを主経路として取得した値を自動で上書きしない。年度、会計範囲、単位、欠測注記、改訂状況、転記位置を確認し、両方の出典、原表記、セル位置、取得時刻、SHA-256を残す。原因を確定できない場合は、不一致または比較不能の事実だけを記録し、どちらかを推測で採用しない。
