# Tier 3: 予算書・決算書など地域固有文書

> **執筆予定**

自治体固有の予算書・決算書 PDF を、人が確認する前提で取り込み、表抽出後の総額照合を通ったデータだけを利用可能にするための設計文書です。PDF の見た目から数値を集めるだけでなく、会計区分、当初・補正・最終、単位、符号、ページを保持し、合計不一致を公開利用の停止条件にします。

## 章立て

- 対象文書の分類と原本登録
- テキスト PDF、画像 PDF、混在 PDF の判定
- 表、脚注、改ページの抽出
- 会計区分と当初・補正・決算の識別
- 円・千円・百万円の単位変換
- △、括弧、マイナス、欠測記号の扱い
- 小計・総計・差引額による `verify_totals`
- 端数処理と表示額・内部額の差
- 総額不一致時の隔離と人手修正
- ページ単位の provenance と再抽出
- SQLite への格納と Obsidian の判断ノート
- 公開前検証ゲート
