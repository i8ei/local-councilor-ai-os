# v0.1.8

## Added

- Added `onboarding/` with read-only environment diagnosis, deterministic plan hashes, safe Vault scaffolding, run manifests, and verification.
- Added strict Obsidian CLI checks for exact Vault path and targeted search connectivity.
- Added namespaced Vault scaffolding for eight work shelves, templates, operating principles, workflows, and incomplete user/council profiles.
- Added permission preflight output and `integrate`, `full`, `components`, and `diagnose` onboarding modes.
- Added automatic Claude Code/Codex discovery, one-time selection when both are available, and client-specific guide, permission, setup, and rerun instructions.
- Added a budget/settlement index preflight that lists official document candidates without downloading documents or creating SQLite.
- Added a machine-readable source registry covering e-Stat, JGrants, and municipal budget/settlement indexes, including live/cache/SQLite/snapshot boundaries.
- Added static-minutes `--dry-run` output with selected and excluded candidate reasons.
- Added optional `--question` fields to minutes and regulations context packs while preserving the search expression separately.

## Fixed

- Static minutes discovery now matches percent-decoded filenames, supporting official pages whose link text contains only a PDF size.
- Municipal fiscal index preflight now inherits budget/settlement context from the page title and primary heading, and prefers human-readable labels for duplicate document links.
- Obvious XLSX binary-float tails are shortened in display `raw_value` while the exact XML value remains in `source_locator`.

## Verified

- Verified the full test entry point and reconciliation gates.
- Verified clean-room onboarding and a real new Obsidian Vault for a Kashima City scenario without modifying the user's existing Vault.
- Verified auto selection, explicit Codex reuse, and actionable Claude Code handoff in the real Kashima City test Vault.
- Verified Kashima City's official budget and settlement index preflight with zero document downloads.
- Verified Kashima City minutes discovery with zero meeting-body/PDF downloads and zero database creation.
- Verified online bootstrap and cache-only offline rebuild for Kashima City, Saga Prefecture, with identical normalized indicator rows.
