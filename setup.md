# セットアップ

> **執筆予定**

この章では、前提となる [`claude-obsidian-setup`](https://github.com/i8ei/claude-obsidian-setup) の規約に沿って基盤環境を用意した後、このリポジトリ固有のディレクトリ、SQLite、プロファイル、権威マップ、取得キャッシュを安全に追加する手順を説明します。現時点では実装CLIが未完成のため、検証されていないコマンドは掲載しません。

## 章構成

- 前提環境と基盤リポジトリ
- 公開領域と内部領域の分離
- プロファイルの初期設定
- 自治体コードの解決
- 全国共通データの取得
- SQLite／FTS5検索層の構築
- `authority_map.yaml` の生成
- Obsidian正本ノートとの接続
- 整合性確認と更新手順
- アンインストールとデータ削除
